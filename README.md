
# Sorting Visualizer

A web-based sorting visualizer built using:

- HTML
- CSS
- JavaScript

Algorithms Implemented:
- Bubble Sort
- Selection Sort
- Insertion Sort

Features:
- Dynamic Array Generation
- Speed Control
- Array Size Control
- Algorithm Complexity Display
- Interactive Animations
